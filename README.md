# Ink & Echo – A Premium Blog Platform

Ink & Echo is a sophisticated, minimalist blog application designed for thoughtful writing and personal reflections. Featuring a clean, modern aesthetic with glassmorphism elements, it provides a seamless experience for both authors and readers.

Designed and Developed by **Mradul Khandewal**.

## ✨ Features

- **Personalized Accounts**: Secure sign-up and login functionality.
- **Thoughtful Reading Experience**: Focused article view designed for maximum legibility.
- **Content Management**: Create, edit, and manage your posts with ease.
- **Draft System**: Save your unfinished ideas and publish them when you're ready.
- **Responsive Design**: Optimized for desktop, tablet, and mobile viewing.
- **Premium Aesthetics**: Built with a curated color palette (Forest Green & Sand) and high-quality typography.

## 🛠️ Technology Stack

- **Backend**: Django 6.0.3 (Python)
- **Database**: PostgreSQL (Production) / SQLite (Development)
- **Styling**: Vanilla CSS with Modern UI principles (Glassmorphism, CSS Variables)
- **Assets**: Whitenoise for static file management, Pillow for image processing.
- **Fonts**: Inter (Sans-serif) & Playfair Display (Serif) via Google Fonts.

## 🚀 Getting Started

### Prerequisites

- Python 3.10+
- pip (Python package manager)

### Installation

1. **Clone the repository**:
   ```bash
   git clone <your-repository-url>
   cd blog_project-main
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Apply migrations**:
   ```bash
   python manage.py migrate
   ```

5. **Run the development server**:
   ```bash
   python manage.py runserver
   ```

6. **Access the application**:
   Open your browser and navigate to `https://blog-project-django-zh3z.onrender.com`.
